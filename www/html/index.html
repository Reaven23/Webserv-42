⚠ Error response tests

Click to test in browser:

  • 404 /files/ghost.txt
  • 403 path traversal attempt

Test via curl: 

# 405 - wrong method
curl -i -X POST http://localhost:8080/

# 413 - body too large
curl -i -X POST \
  --data-binary @<(head -c 3000000 /dev/zero) \
  http://localhost:8080/upload

# 404 - missing file
curl -i -X DELETE \
  http://localhost:8080/files/nope.txt
Restricted server :8085

🔒 Body limit: 1 KB

This server has client_max_body_size 1000. Any POST larger than 1 KB is rejected. 

# Should be rejected (413)
curl -i -X POST --data-binary <@(head -c 2000 /dev/zero) http://localhost:8085/limited

# Should work
curl -i http://localhost:8085/

📋 curl cheatsheet

# ── GET ──
curl -i http://localhost:8080/
curl -i http://localhost:8080/files/

# ── POST upload ──
curl -i -X POST \
  -F "file=@photo.jpg" \
  http://localhost:8080/upload

# ── DELETE ──
curl -i -X DELETE http://localhost:8080/files/photo.jpg

# ── CGI ──
curl -i http://localhost:8084/py-cgi/echo.py?key=value
curl -i http://localhost:8084/js-cgi/clearUpload.js

# ── Redirects (show headers, don't follow) ──
curl -i http://localhost:8081/old-page
curl -i http://localhost:8081/temp-page

# ── Multiple servers ──
curl -i http://localhost:8080/   # main-site
curl -i http://localhost:8081/   # redirects
curl -i http://localhost:8082/   # read-only
curl -i http://localhost:8083/   # uploads-only
curl -i http://localhost:8084/   # cgi-server
curl -i http://localhost:8085/   # restricted
